# Diploma Work Landing

Лендинг бакалаврської роботи на тему **«Мобільний застосунок для підбору туристичних турів»**.

## Автор
- Леонід Раєв (Leonid Raiev)
- Група ІН-22/2
- Сумський державний університет

## Опис
Цей репозиторій містить лендинг, створений для презентації бакалаврської роботи, присвяченої розробці мобільного застосунку для підбору туристичних турів.

## Посилання
- Основний репозиторій проєкту: https://github.com/tekamolo123/diploma_work_travel_app
- Пояснювальна записка: https://github.com/tekamolo123/diploma_work_travel_app/blob/main/README.md

## Структура
- `index.html` — головна сторінка лендингу
- `css/styles.css` — стилі сайту
- `robots.txt` — інструкції для пошукових систем
- `sitemap.xml` — карта сайту
- `assets/` — зображення та іконки

## Статус
На поточному етапі створено базову структуру лендингу.
