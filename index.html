Бакалаврська робота

Мобільний застосунок для підбору туристичних турів

Лендинг проєкту, присвяченого розробці мобільного застосунку для пошуку, перегляду та вибору туристичних турів.

Про проєкт

Даний лендинг презентує бакалаврську роботу, пов’язану з розробкою мобільного застосунку для підбору туристичних турів. Основна мета проєкту — створити зручний інструмент для користувачів, який дозволяє переглядати доступні тури, знаходити потрібні пропозиції та отримувати детальну інформацію про подорожі.

Статус етапу

На поточному етапі створено базову структуру лендингу: головну HTML-сторінку, файл стилів, технічні файли для пошукових систем та каталоги для майбутніх зображень і іконок.
